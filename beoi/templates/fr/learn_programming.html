L'informatique est un domaine de défis permanents, que ce soit dans le développement d'un site web d'un festival ayant quelques milliers de visiteurs, de l'application qu'il vous manque pour smartphone ou de celle pour piloter votre nouvel robot-aspirateur. L'informatique couvre de très nombreux domaines de la vie courante et peut être l'occasion de mettre à l'épreuve votre créativité et réflexion. Et si vous étiez le fondateur du Facebook de demain ?

Pour pouvoir créer à partir d'un ordinateur, tout commence par la programmation. Malheureusement, ça ne s'apprend pas à l'école ou que très superficiellement. Il est souvent nécessaire d'apprendre par vous même, et comme tout sport ou instrument du musique, il sera probablement nécessaire d'y passer un peu de temps et s'exercer régulièrement.

Pour ceux qui ne savent par où commencer le programmation, vous trouverez ici quelques conseils. La plupart de ressources extérieures sont en anglais, mais c'est très souvent de l'anglais simple et après tout, cela vous permettra d'améliorer votre anglais !

Le site Khan Academy offre une bonne introduction à la programmation sans avoir besoin de connaitre au préalable un langage de programmation.

La finale de l'olympiade belge ainsi que l'olympiade internationale acceptent les deux languages suivants : C++ and Java.

Le site d'entrainement de la délégation française à l'Olympiade Internationale d'Informatique comporte de nombreux cours permettant d'apprendre à programmer et de progresser rapidement. On peut y trouver de petits problèmes et de gros exercices allant d'un niveau débutant à (très) expérimenté.

Java

Un moyen facile de commencer à programmer en Java est d'utiliser l'environnement de dévelopement "Eclipse". Commencer par installer Eclipse Standard Edition.

Un tutoriel officiel pour apprendre le Java est disponible ici.

C++

Le C++ est un langage de programmation très puissant mais dans lequel il n'est pas très facile de se lancer. Un programme C++ est souvent plus long qu'un programme Python mais peut être bien plus rapide et efficace. Le C++ est donc beaucoup utilisé dans les applications ayant des besoins temps réels tels que des jeux (Doom III, Starcraft, ...), navigateurs web, logiciels professionnels de traitement d'images (Photoshop, Maya, ... ) et même dans les coulisses de Facebook et Amazon.

Il existe quelques bons sites sur lesquels vous pouvez apprendre le C++ et vous exercer.

  • Si vous avez déjà programmé, vous pouvez commencer le C++ via le tutoriel de cplusplus.com.
  • Une présentation plus détaillée du C++ se trouve sur LearnCPP.com. Le chapitre 0 vous aidera à installer les outils nécessaires sur votre propre ordinateur. Les sept premiers chapitres vous enseigneront les rudiments du langage, les suivants couvrent des sujets plus avancés.
  • Le site Programmr a également de nombreux exercices disponibles en C++.
  • Pour installer le compilateur C++ (GCC) sous Windows, suivez la video tutorielle suivant https://www.youtube.com/watch?v=YLkFKcHvELc. GCC est le compilateur utilisé lors des épreuves beOI et IOI. Sous Mac (OSX) et Linux, GCC est intégré ou peut être installé très facilement.

Compétition de programmation

Une olympiade est plus que de la simple programmation : votre solution doit également être efficace et écrite rapidement. Une bonne connaissance en algorithmique est indispensable pour résoudre les problèmes les plus complexes. En effet, connaître des solutions aux problèmes courants vous permettra d'appliquer ces techniques à de nouveaux problèmes originaux. De nombreux sites web vous permettent de vous former à l'algorithmique, entre autres le site de l'USA Computing Olympiad (nécessite de s'enregistrer).

Il n'y a qu'une seule façon d'améliorer dans les concours de programmation: s'entraîner ! Heureusement, il existe de nombreux concours internationaux auxquels vous pouvez participer gratuitement (notre liste de concours disponibles). Nous vous recommandons particulièrement CodeForces qui propose des concours de qualité très régulièrement.
